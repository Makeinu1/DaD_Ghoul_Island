# リプレイ小説 文体統一ガイドライン

**作成日:** 2025-12-31
**目的:** 第1～11回のリプレイ小説全体で文体を統一し、読みやすく高品質なリプレイを提供する

---

## 🎯 基本方針

### 1. 統一された語り手の視点
- **三人称客観視点**を基本とする
- プレイヤーログの一人称描写は三人称に変換
- 地の文は「である調」で統一（例外なし）

### 2. 文体の一貫性
- 硬すぎず、柔らかすぎず、バランスの取れた文体
- 読み物として楽しめる「小説的な」表現
- プレイヤーログの生々しさを残しつつ、洗練された文章に

---

## 📝 地の文のルール

### ✅ 推奨される文体

**基本形: 「である調」統一**
```
良い例:
ノクシルは港に到着した。腰の袋から音が聞こえ、懐かしそうな顔でトーテムを取り出す。

シャルルは書類から顔を上げ、冷静に告げた。ディードリドは深呼吸をして、それから口を開く。

巡礼者は星々が煌めく次元界を漂っていた。馴染みの宇宙カレー屋で食事を摂っていると、主であるシュブ＝ニグラスが現れる。
```

**避けるべき文体:**
```
悪い例（ですます調の混在）:
ノクシルは港に到着しました。腰の袋から音が聞こえ、懐かしそうな顔でトーテムを取り出す。

悪い例（口語的すぎる）:
ノクシルが港に着いた。腰の袋からなんか音が聞こえてきて、懐かしそうにトーテムを取り出すのであった。

悪い例（硬すぎる）:
ノクシル・グリムホルトはレサンテ港に到着せり。腰部に装着せる袋より音響を感知し、トーテムを取り出すに至る。
```

### 文章のリズム
- **短文と長文を適切に配置**する
- 緊迫した場面: 短文中心（例: 「剣が閃く。敵が倒れる。戦いは続く」）
- 情景描写: 長文も可（例: 「夕暮れの港に、オレンジ色の光が差し込み、波の音が静かに響いていた」）

### 描写の濃淡
**濃い描写（重要シーン）:**
- 戦闘の重要な判定
- キャラクターの心情が動く瞬間
- ストーリーの転換点
- NPCとの重要な会話

**薄い描写（移動・日常）:**
- 移動シーン
- ルーティン作業
- 雑談（重要でない部分）

## 🎭 キャラクター描写のルール

### 内面描写の扱い

**✅ 許可される内面描写:**
- プレイヤーログに明記されている内面
- シャルル日誌、ノクシルログの心情描写

**例（シャルル日誌より）:**
```
かつての私なら、自らの使命を優先し、巡礼者を捨て置いたに違いない。だが今の私は、私であるために"義"を通さねばならぬ。

→ そのまま採用可能
```

**❌ 避けるべき創作的内面描写:**
```
悪い例:
ノクシルは内心で不安を感じていた。この島には何かがある。それを確信していた。

（ノクシルログにこの記述がない場合は創作になるため不可）
```

### 行動描写の表現

**✅ 推奨:**
```
ノクシルは剣を抜き、構えた。
シャルルは駆け寄り、声をかける。
巡礼者はゆっくりと首を傾げた。
```

**❌ 避ける:**
```
ノクシルは剣をシャキーンと抜いた。（擬音語の乱用）
シャルルはダッシュで駆け寄った。（カタカナ口語）
巡礼者は不思議そうに首を傾げたのであった。（過剰な推測）
```

## 🗣️ セリフの扱い（別ガイドラインと併用）

### セリフの基本形式

**統一フォーマット:**
```markdown
**キャラクター名:**
「セリフの内容」
```

**例:**
```markdown
**ノクシル:**
「問題ない。すぐ行こう」

**シャルル:**
「大事の前の小事だ。今すぐ行く」

**巡礼者:**
「人の子は、私を巡礼者と呼ぶ」
```

### 複数行のセリフ

```markdown
**センサ:**
「ノクシル、元気にしているか」

ノクシルは頷く。

「最近はどうだ？　どんな冒険をしている？」
```

## 🔧 文体修正の実践例

### 修正前（第5回より抜粋）
```
ノクシルとセンサは街中へ行き、泊まれる宿を探すことに。

拘置所を出た後、光の女神の像がある広場の前を通り過ぎるノクシルとセンサ。

街の広間で女神像を改めて観察した。一見おかしな像に見えた、それは確かにこの街を護っているようで、謝罪の意も込め像に祈りを捧げた。
```

### 修正後
```
ノクシルとセンサは街中へ向かい、泊まれる宿を探すことにした。

拘置所を出た後、光の女神の像がある広場を通り過ぎる。二人は足を止め、改めて女神像を観察した。

一見奇妙な造形に見えるが、確かにこの街を守護しているような威厳を感じる。謝罪の意を込め、ノクシルは静かに祈りを捧げた。
```

**修正ポイント:**
- 「〜ことに。」→「〜ことにした。」（文末統一）
- 主語の明確化（「二人は」「ノクシルは」）
- 文のつながりを自然に
- 「である調」の徹底

## 📊 チェックリスト

修正時に以下を確認:

- [ ] 地の文が「である調」で統一されているか
- [ ] 「ですます調」が混在していないか
- [ ] 主語が明確か（主語不明の文がないか）
- [ ] 短文・長文のバランスが取れているか
- [ ] 重要シーンは濃く、移動シーンは薄く描写されているか
- [ ] キャラクターの内面描写がプレイヤーログに基づいているか
- [ ] セリフのフォーマットが統一されているか
- [ ] 口語的すぎる表現、硬すぎる表現がないか
- [ ] 一次データ（プレイヤーログ）の内容と矛盾していないか

## 🎨 文体のトーン

### 目指すべき雰囲気
- **親しみやすさ**: 内輪ネタやユーモアは残す
- **読みやすさ**: スムーズに読める自然な日本語
- **臨場感**: セッションの楽しさが伝わる描写
- **統一感**: 全11回を通して同じ語り手が語っているように

### 避けるべきトーン
- **堅苦しすぎる**: 論文調、古文調
- **幼稚すぎる**: 絵本調、過剰な擬音
- **冗長すぎる**: 同じことを何度も繰り返す
- **説明的すぎる**: ゲーム用語の羅列、メタ発言の過多

## 📚 参考: 文体のグレード

### グレードA（目標）
```
夕暮れのレサンテ港に、ノクシルの姿があった。腰の袋から聞こえる音に、彼は懐かしそうな表情を浮かべる。半年ぶりの連絡だ。トーテムを取り出すと、センサ船長の声が響いた。

「ノクシル、元気にしているか」

「ああ、そうだな。半年ぶりだよ」

二人の会話は、まるで昨日も会っていたかのように自然だった。
```

### グレードB（許容範囲）
```
夕暮れのレサンテ港にノクシルがいた。腰の袋から音が聞こえ、懐かしそうにトーテムを取り出す。センサ船長の声がした。

「ノクシル、元気にしているか」
「ああ、半年ぶりだな」

二人は半年ぶりの再会だった。
```

### グレードC（要修正）
```
夕暮れのレサンテ港にノクシルがいました。腰の袋から音が聞こえてきて、懐かしそうな顔をしてトーテムを取り出すのであった。するとセンサ船長の声が聞こえてきた。

「ノクシル、元気？」
「うん、半年ぶりだね」

なんと二人は半年ぶりの再会だったのだ！
```

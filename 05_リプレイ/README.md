# リプレイ作成ガイド

このディレクトリには、D&D 食屍鬼島キャンペーンのリプレイが保存されます。

## ファイル構成

- `リプレイテンプレート.md` - リプレイ作成用のテンプレート
- `第1回_レサンテ港からの旅立ち.md` - 第1回セッションのリプレイサンプル
- `第X回_タイトル.md` - 各セッションのリプレイ

## リプレイ作成の手順

### 1. プレイヤーログの確認

各プレイヤーのログを確認します：
- `04_プレイヤーログ/巡礼者_PLチョモラン/`
- `04_プレイヤーログ/ノクシル_PLたつや/`
- `04_プレイヤーログ/シャルル_PLまけいぬ/`

### 2. 音声ログの参照（参考程度）

`07_セッション音声ログ/` のSRTファイルから、特徴的な台詞や重要な会話を抽出します。

**注意点：**
- 音声ログには雑談も多く含まれているため、**あくまで参考程度**に使用
- 録音失敗している回もあるため、必ずしも全てのセッションがあるわけではない
- 特徴的な台詞や印象的な会話を拾うことを重視
- プレイヤーログと矛盾する場合は、プレイヤーログを優先

**音声ログから抽出すべき情報：**
- キャラクターの特徴的な台詞
- NPCとの重要な会話
- プレイヤー間の印象的なやり取り
- DMの描写で印象的な部分

### 3. シャルルストーリーの参照

`03_プレイヤー設定/シャルルストーリー/` のファイルから、シャルルの背景情報を参照します。

**含まれる情報：**
- DMからのシャルルの設定
- ルンジャタとの過去のエピソード（慈悲の澱）
- 深きものとの遭遇経験（誘いの歌声）
- キャラクターの深層心理や動機

**活用方法：**
- シャルルの行動の背景を理解する
- リプレイ内でシャルルの内面描写を補強する
- 伏線や謎の理解に役立てる

### 4. テンプレートの使用

`リプレイテンプレート.md`をコピーして、新しいリプレイファイルを作成します。

### 5. 情報の統合

各プレイヤーのログから以下の情報を抽出して統合します：

- **時系列の流れ** - 巡礼者のログが主軸になりやすい
- **キャラクターの内面** - シャルルとノクシルのログから感情や思考を引用
- **重要な出来事** - 戦闘、判定、NPCとの会話など
- **伏線や謎** - 次回へのつながり
- **音声ログからの台詞** - 特徴的な会話を適切に配置
- **背景情報** - シャルルストーリーなどから補足情報を追加

### 4. リプレイの書き方のコツ

- **シーンごとに区切る** - 時系列順に、シーンごとに整理
- **DMの描写とプレイヤーの行動を分ける** - 読みやすさを重視
- **キャラクターの声を活かす** - 各プレイヤーのログから会話や思考を引用
- **判定結果を簡潔に** - 重要な判定のみ詳細に、その他は簡潔に
- **次回への伏線を明確に** - 読者が続きを読みたくなるように

## Cursorでの活用方法

1. **コードベース検索** - 特定のセッションや出来事を検索
   - 「第X回目の嵐のシーン」など、セマンティック検索で該当箇所を探す
2. **ファイル参照** - プレイヤーログを参照しながらリプレイを作成
   - 複数のログを並べて比較しながら統合
3. **音声ログの検索** - 特定の台詞や会話を検索
   - キャラクター名やキーワードでgrep検索
   - ただし雑談が多いため、重要な部分のみ抽出
4. **背景情報の参照** - シャルルストーリーなどから設定を確認
   - キャラクターの行動の動機を理解
5. **一貫性の確保** - 過去のリプレイを参照して、文体や形式を統一

## 音声ログの活用例

音声ログ（SRT形式）から重要な台詞を抽出する際の例：

```
例：第1回目の音声ログから
- 「ノクシル元気にしているかと」（センサ船長）
- 「船長がついに船長になったのか」（ノクシル）
- 「あなたが私のことを船長って呼んでくれてるのはも嬉しかったけどや、これで本当に私を船長になることはできたんだ」（センサ船長）
```

これらの台詞をリプレイに組み込むことで、キャラクターの関係性や感情がより鮮明になります。

## シャルルストーリーの活用例

シャルルストーリーから得られる情報をリプレイに反映する例：

- **ルンジャタとの関係** - 過去に共に不正を暴いた経験があるため、シャルルがルンジャタの失踪に強い関心を持つ理由が明確になる
- **深きものへの恐怖** - 過去の経験から、深きものに対して特別な警戒心を持っている
- **不正への怒り** - ディクスター司祭の件など、不正に対する強い反応の背景が理解できる

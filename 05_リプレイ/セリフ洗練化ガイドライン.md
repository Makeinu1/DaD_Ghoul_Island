# リプレイ小説 セリフ洗練化ガイドライン

**作成日:** 2025-12-31
**目的:** 音声ログからの自動文字起こしによる不自然なセリフを、リプレイ小説に適した自然な日本語に洗練する

---

## 🎯 基本方針

### セリフの2つの情報源

1. **音声ログ（SRT）**: 自動文字起こし → 精度が低い、口語的すぎる
2. **プレイヤーログ**: PLが記録したセリフ → より信頼性が高い

**原則: プレイヤーログ > 音声ログ**

---

## 🔧 音声ログセリフの問題点と修正方法

### 問題1: 自動文字起こしの誤変換

**❌ 音声ログのまま:**
```
「そのとり、そのとりあってるっていうのが本当ならば」
「いや、あなたには隠し事ができないな」（文脈不明）
「コツが溢れる人ですね」（骨→コツの誤変換）
```

**✅ 修正後:**
```
「その取り合いがあったというのが本当ならば」
「あなたには隠し事ができないな」
「骨が溢れる人ですね」
```

### 問題2: 過剰な言い淀み・フィラー

**❌ 音声ログのまま:**
```
「えっと、その、まあ、およそ1000年前から500年前の長期間にわたる戦争だったんですが」
「いや、いや、あなたが私のことを船長って呼んでくれてるのは嬉しかったけどや」
```

**✅ 修正後:**
```
「およそ1000年前から500年前の長期間にわたる戦争だったのですが」
「あなたが私のことを船長と呼んでくれるのは嬉しかったが」
```

**修正ポイント:**
- 「えっと」「その」「まあ」→ 削除（意味のあるもの以外）
- 「いや、いや」→ 「いや」に統一
- 「〜けどや」→ 「〜が」に修正

### 問題3: 口語的すぎる表現

**❌ 音声ログのまま:**
```
「かかあが今夜峠を越えられるかがかかっているんだ」
「なんだ、そんなことかい？　いや、それはいい」
「どうだろうか？」
```

**✅ 修正後:**
```
「妻が今夜峠を越えられるかがかかっているんだ」
「それならば問題ない」
「どうだろうか」（疑問符は残す）
```

**修正ポイント:**
- 「かかあ」→ 「妻」（文脈によっては「妻」「家内」）
- 「〜かい？」→ 「〜か」
- 不要な「いや」を削除

### 問題4: 疑問符・感嘆符の乱用

**❌ 音声ログのまま:**
```
「南区の医師、ミナミにこの薬を届けてくれないか！　かかあが今夜峠を越えられるかがかかっているんだ！」
「なんだと！」
「どうだろうか？？」
```

**✅ 修正後:**
```
「南区の医師、ミナミにこの薬を届けてくれないか。妻が今夜峠を越えられるかがかかっているんだ」
「なんだと！」（強調したい場合のみ残す）
「どうだろうか」
```

**修正ポイント:**
- 疑問符・感嘆符は強調したいセリフのみ
- 「！！」「？？」の重複は不可
- 基本は句点「。」で終わる

## 📝 キャラクター別セリフの特徴

### 巡礼者

**特徴:**
- 異形だが純粋
- 人間の文化に興味を持つ
- 直接的で率直な物言い
- やや古風な話し方

**✅ 良いセリフ:**
```
「人の子は、私を巡礼者と呼ぶ」
「話が分かるな」
「文化とは興味深い」
「それは矜持というものか」
```

**❌ 避けるべき:**
```
「えっと、私のことは巡礼者って呼んでくださいね」（丁寧すぎる）
「マジで？」（現代口語）
「わかんない」（砕けすぎ）
```

### ノクシル

**特徴:**
- 寡黙でクール
- 「問題ない」が口癖
- 必要最小限の言葉
- レンジャーとしての観察眼

**✅ 良いセリフ:**
```
「問題ない」
「ああ、そうだな」
「困った時はお互い様だな」
「楽しみにしてるよ、船長」
```

**❌ 避けるべき:**
```
「全然大丈夫ですよ！」（明るすぎる）
「えー、どうしようかなー」（優柔不断）
「マジで問題ないっす」（軽すぎる）
```

### シャルル

**特徴:**
- パッション溢れる
- 正義感が強い
- やや古風で格調高い言い回し
- 「義」「矜持」などの言葉を使う

**✅ 良いセリフ:**
```
「大事の前の小事だ」
「我々監査官は優雅に振る舞うものだ」
「それは義に反する」
「我が剣が許さん」
```

**❌ 避けるべき:**
```
「まあ、いいんじゃない？」（軽すぎる）
「ヤバいね」（現代口語）
「どうしよっかな」（格調が低い）
```

### センサ船長

**特徴:**
- 明るく親しみやすい
- ノクシルへの信頼と友情
- 夢を実現した喜び

**✅ 良いセリフ:**
```
「ノクシル、元気にしているか」
「ついに手に入ったんだ。私の船が」
「楽しみにしてるよ」
```

**❌ 避けるべき:**
```
「ノクシルさーん、元気ー？」（軽すぎる）
「船、マジでゲットしたわ」（現代口語）
```

## 🎨 セリフ修正の実践例

### 例1: 音声ログから洗練化

**音声ログ（第11回より）:**
```
「まあ、ま、それでしたら別にいいですか。私もあの人のこと嫌いですし」
```

**修正後:**
```
「それでしたら別にいいですよ。私もあの人のこと、嫌いですし」
```

**修正ポイント:**
- 「まあ、ま」削除
- 「いいですか」→「いいですよ」（肯定に変更）
- 読点の追加で読みやすく

---

### 例2: プレイヤーログを優先

**音声ログ:**
```
「えっと、その、グールとの協定の話なんですが」
```

**プレイヤーログ（巡礼者ログ）:**
```
マッテオはグールとの協定について語った
```

**修正後:**
```
「グールと人間の不可侵協定の話です」
```

**修正ポイント:**
- プレイヤーログの内容をベースに
- フィラー削除
- 明確な表現に変更

---

### 例3: キャラクターの個性を活かす

**音声ログ:**
```
「なんか、そういうのあるんだよね」
```

**キャラクター: 巡礼者の場合:**
```
「そのようなものがあるのか」
```

**キャラクター: ノクシルの場合:**
```
「そうか」
```

**キャラクター: シャルルの場合:**
```
「そのような事情があったのか」
```

## 📋 セリフ修正チェックリスト

修正時に以下を確認:

- [ ] プレイヤーログとの整合性を確認したか
- [ ] 自動文字起こしの誤変換を修正したか
- [ ] 不要なフィラー（「えっと」「その」「まあ」）を削除したか
- [ ] 口語的すぎる表現を修正したか
- [ ] 疑問符・感嘆符は適切に使用されているか
- [ ] キャラクターの口調・個性が反映されているか
- [ ] セリフの長さは適切か（長すぎず、短すぎず）
- [ ] 前後の文脈と矛盾していないか

## 🚫 絶対に避けるべき表現

### 現代口語・ネットスラング
```
❌ 「マジで」「ヤバい」「キモい」「ウケる」「エモい」
❌ 「草」「www」「orz」
❌ 「それな」「わかりみが深い」
```

### 過剰な擬音語
```
❌ 「ドカーン」「ガシャーン」「シャキーン」
❌ 「ドキドキ」「ワクワク」「イライラ」（地の文ならOK）
```

### 不自然な敬語
```
❌ 「〜でございますぅ」
❌ 「〜ですぅ」「〜ますぅ」
```

## ✅ 推奨される表現

### 時代感のある言い回し（ファンタジー世界）
```
✅ 「〜であろう」「〜だろうか」
✅ 「〜ではないか」「〜ではなかろうか」
✅ 「〜というわけか」
✅ 「そうか」「なるほど」「確かに」
```

### 自然な会話表現
```
✅ 「ああ、そうだな」
✅ 「問題ない」
✅ 「それは良かった」
✅ 「なんと」「ほう」
```

## 🎭 セリフの長さとリズム

### 短いセリフ（緊迫した場面）
```
✅ 「逃げろ！」
✅ 「待て」
✅ 「問題ない」
```

### 中程度のセリフ（通常の会話）
```
✅ 「それは確かに不可解だな」
✅ 「船長の話は本当だったか」
```

### 長いセリフ（説明・演説）
```
✅ 「およそ1000年前から500年前の長期間にわたる戦争だったのですが、当時、島民は追い詰められていました」

※ 長いセリフは地の文で分割することも検討
```

## 📖 修正前後の比較例

### 第1回 センサ船長のセリフ

**修正前（音声ログベース）:**
```
「いや、あなたには隠し事ができないな。実はな、ついに手に入ったんだ。私の船が」

「そう、そうなんだ。いや、いや、あなたが私のことを船長って呼んでくれてるのは嬉しかったけどや、これで本当に私は船長になることができたんだ」
```

**修正後:**
```
「あなたには隠し事ができないな。実は、ついに手に入ったんだ。私の船が」

「そうなんだ。あなたが私のことを船長と呼んでくれるのは嬉しかったが、これで本当に船長になることができた」
```

## 🎯 最終目標

**読者が自然に読める、キャラクターの個性が光るセリフ**

- プレイヤーとDMが読んで「このセリフ、確かに言ったな」と思える
- 初見の読者が読んでも違和感がない
- キャラクターの声が聞こえてくるような臨場感

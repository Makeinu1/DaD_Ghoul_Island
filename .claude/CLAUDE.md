## 最重要ルール

### データの優先順位

```
一次データ > 公式設定 > 二次データ（参考のみ）
```

1. **04_プレイヤーログ/** - 最優先（巡礼者ログが最も網羅的）
2. **07_セッション音声ログ/** - 特徴的な台詞のみ抽出
3. **03_プレイヤー設定/** - 公式設定
4. **05_リプレイ/** - ⚠️参考のみ（誤りの可能性）
5. **08_LLMによるキャラ評/** - ⚠️参考のみ（誤りの可能性）

### 重要な注意事項

- **既存キャラ評を鵜呑みにしない**（第12回で複数の誤りを修正）
- **話者エラーに注意**（自分自身を呼ぶ発言がないか確認）
- **デプロイ後は必ずURLアクセスして確認**

## クイックプロンプト

### リプレイ初稿作成

```
第X回のリプレイ小説を執筆してください。

【参照必須】
- 04_プレイヤーログ/巡礼者_PLチョモラン/第X回目.txt
- 04_プレイヤーログ/シャルル_PLまけいぬ/第X回目_*.txt（あれば）
- 07_セッション音声ログ/第X回目-*.srt

【ガイドライン】
- .claude/rules/03_replay-guidelines.md
- 05_リプレイ/_ガイドライン/リプレイテンプレート.md
```

### キャラ評更新

```
第X回のセッションを踏まえて、キャラクター評価を更新してください。

【注意】
- 一次データを最優先で参照
- 既存の記述を無批判に引き継がない
```

### Webデプロイ

```
第X回のリプレイをWebサイトに追加してデプロイしてください。

【チェックリスト】
- .claude/rules/07_web-deployment.md を参照
```

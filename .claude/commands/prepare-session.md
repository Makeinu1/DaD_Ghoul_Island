# 新規セッション準備

第$ARGUMENTS回のセッションに向けた準備を行います。

## 実行内容

### 1. フォルダ確認

以下のフォルダが存在するか確認:

- `04_プレイヤーログ/巡礼者_PLチョモラン/`
- `04_プレイヤーログ/シャルル_PLまけいぬ/`
- `04_プレイヤーログ/ノクシル_PLたつや/`
- `07_セッション音声ログ/`

### 2. 作業フォルダ準備

`05_リプレイ/作業中/` フォルダが存在するか確認

### 3. チェックリスト表示

@.claude/rules/09_session-preparation.md

## セッション後の素材配置

セッション終了後、以下のファイルを配置してください:

```
04_プレイヤーログ/巡礼者_PLチョモラン/第$ARGUMENTS回目.txt
04_プレイヤーログ/シャルル_PLまけいぬ/第$ARGUMENTS回目_メモ.txt（任意）
07_セッション音声ログ/第$ARGUMENTS回目-1.srt
```

## 次のステップ

素材が揃ったら:

```
/create-replay $ARGUMENTS
```

でリプレイ作成を開始できます。

## 前回の振り返り

前回（第$((ARGUMENTS - 1))回）のリプレイを読み返すことを推奨:

```
05_リプレイ/完成版/第$((ARGUMENTS - 1))回_*.md
```

伏線や未解決事項を把握しておくと、今回のセッションの理解が深まります。

# キャラクター評価更新

最新セッションの内容を踏まえて、PCおよびNPCのキャラクター評価を更新します。

## 更新対象

### PC（プレイヤーキャラクター）

- `08_LLMによるキャラ評/巡礼者_キャラクター評.md`
- `08_LLMによるキャラ評/ノクシル・グリムホルト_キャラクター評.md`
- `08_LLMによるキャラ評/シャルル・ビアトリス_キャラクター評.md`

### NPC（登場したNPCのみ）

- `08_LLMによるキャラ評/NPC_*_キャラクター評.md`

## 参照すべきファイル

1. **完成版リプレイ**: `05_リプレイ/完成版/第X回_*.md`
2. **一次データ**: `04_プレイヤーログ/` 配下のログ
3. **既存キャラ評**: `08_LLMによるキャラ評/` 配下

## 重要な注意事項

@.claude/rules/02_data-priority.md
@.claude/rules/08_troubleshooting.md

- **既存の記述を無批判に引き継がない**
- **一次データと矛盾する場合は一次データを優先**
- **バージョン番号を更新** (Ver.1.1 → Ver.1.2 など)

## 更新すべき項目

1. **新しい行動パターン**: 今回のセッションで見られた特徴的な行動
2. **新しい能力/スキル**: 使用した呪文、技能、特殊能力
3. **関係性の変化**: 他のキャラクターとの関係の進展
4. **心理描写**: プレイヤーログから読み取れる内面
5. **既知の誤りの修正**: 過去の誤記載があれば修正

# プロジェクト概要

D&D 5e版とクトゥルフの呼び声TRPGを混ぜたキャンペーン「食屍鬼島」のリプレイ小説作成プロジェクト。

## 基本情報

- **DM**: モクモク
- **プレイヤー**:
  - PL1（チョモラン）: 巡礼者
  - PL2（タツヤ）: ノクシル・グリムホルト
  - PL3（まけいぬ）: シャルル・ビアトリス
- **システム**: D&D 5e/2024 + クトゥルフの呼び声TRPG（混合ルール）
- **セッション数**: 第1回～第12回（継続中）

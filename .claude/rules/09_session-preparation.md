# セッション準備（Phase 0）チェックリスト

新規セッション実施前後の素材準備ガイド。

---

## セッション前（任意）

- [ ] 前回のリプレイを読み返す
- [ ] 前回の伏線・未解決事項を確認
- [ ] キャラ評を確認して最新状態を把握

---

## セッション実施直後（当日中）

### データ収集確認

- [ ] **巡礼者ログ**: PLチョモランがログを記録したか確認
- [ ] **音声録音**: 録音が正常に完了したか確認（ファイルサイズ > 0）
- [ ] **シャルル日誌/メモ**: PLまけいぬがメモを残す予定か確認
- [ ] **ノクシルログ**: PLタツヤがログを残す予定か確認（第1,5,10回のみ実績あり）

### ファイル配置

```bash
# プレイヤーログ
04_プレイヤーログ/巡礼者_PLチョモラン/第X回目.txt
04_プレイヤーログ/シャルル_PLまけいぬ/第X回目_メモ.txt（または _シャルル日誌.txt）

# 音声ログ
07_セッション音声ログ/第X回目-1.srt
```

---

## リプレイ作成前（1-2日以内）

### 素材の確認

- [ ] 巡礼者ログを読んで内容を把握
- [ ] 音声ログ（SRT）をざっと確認
- [ ] 特徴的な台詞や重要シーンをメモ
- [ ] 新規NPC/新規設定があれば記録

### フォルダ準備

- [ ] `05_リプレイ/作業中/` に作業ファイルを作成
- [ ] 必要に応じてキャラ評テンプレートを準備

---

## 新規NPC登場時

### キャラ評の初期化

1. `08_LLMによるキャラ評/NPC_（キャラ名）_キャラクター評.md` を作成
2. テンプレート: `05_リプレイ/テンプレート/キャラ評テンプレート_NPC.md`

### Webサイト対応（必要に応じて）

1. イラストがあれば `06_イラスト/` と `09_Webサイト/docs/images/` に配置
2. `09_Webサイト/docs/characters/` にキャラページ作成
3. `09_Webサイト/mkdocs.yml` のnav更新

---

## カスタムコマンド

以下のコマンドで自動化可能:

```bash
/prepare-session 13  # 第13回の準備
/create-replay 13    # 第13回のリプレイ作成
/update-chara-eval   # キャラ評更新
/deploy-web          # Webサイトデプロイ
```

# キャラクター紹介

食屍鬼島を冒険する三人の冒険者たちの紹介です。

---

## 巡礼者

![巡礼者](images/巡礼者.png){ width="400" }

**プレイヤー**: チョモラン
**種族**: シュブ＝ニグラスの信者、次元界を漂う存在
**外見**: 黒くて不気味な姿。巨大な蹄の足跡を残すほどの体躯

### 性格・特徴

異形の存在でありながら、純粋で邪気のない心を持つ。シュブ＝ニグラス（黒き子山羊の母）に仕える存在として、ファルジーン島の異変を調査するために旅立つ。

人間の「矜持」や「文化」に強い興味を持ち、仲間との会話を通じて人間の価値観を学ぶことを楽しんでいる。戦士として優れた実力を持ち、シャルルとノクシルを「実力のある戦士」として認め、協力を求める。

### 特徴的な言動

- 「お前たちの造形がまた一つ深まった。ますます、お前たちとの旅が楽しくなってきたよ」
  → 人間の価値観を学ぶことを楽しんでいる

- 「それは汚れているのではない、ただ自分にとって不都合だから敵と見做しているだけだ」
  → 実用的で、しかし無慈悲ではない判断力

- 「もう眠い。お前もノクシルももう寝ろ」
  → 仲間を気遣うが、人間とは異なる時間感覚を持つ

### 戦闘スタイル

クラブやオクトパスなどの異形を切り伏せる戦闘能力を持つ。異形ゆえの特殊な力を持ち、同胞である黒き子山羊の存在を感じ取ることができる。

## ノクシル・グリムホルト

![ノクシル](images/ノクシル.png){ width="400" }

**プレイヤー**: タツヤ
**種族**: エルフ
**クラス**: レンジャー
**外見**: エルフらしく若い見た目だが、実際の年齢は不明

### 性格・特徴

流浪のレンジャーとして各地を放浪し、人知れず善行を行っている。寡黙でクールな性格だが、仲間との絆を大切にする。エルフとしての長い寿命と、人間との時間感覚の違いを実感しながらも、人間との友情を保っている。

センサ船長とは半年ごとにトーテムを通して連絡を取り合う深い絆で結ばれている。観察力と洞察力に優れ、仲間の内心の不安に気づき、さりげなく気遣う。

### 特徴的な言動

- 「船長がついに船長になったのか」
  → センサ船長の成長を心から喜ぶ

- 「無事に乗り切れば良い冒険譚になるだろうさ」
  → 前向きで楽観的な姿勢

- 「問題ない」
  → 口癖。落ち着いた性格を表す

- 「エルフは寝る必要がないんだよ」
  → エルフとしての特性

### 戦闘スタイル

二刀使いの戦闘スタイルで、圧倒的な火力を発揮する。オクトパスにクリティカルを決めるなど、戦闘における実力は群を抜いている。エルフとしての優雅さと、レンジャーとしての高い身体能力を併せ持つ。

## シャルル・ビアトリス

![シャルル](images/シャルル.png){ width="400" }

**プレイヤー**: まけいぬ
**種族**: ヒューマン
**クラス**: イルメイターの信徒（クレリック）
**外見**: 戦争孤児として育った過去を持つ、力強い女性

### 性格・特徴

イルメイターの信徒として、不正と不実が大嫌いな性格。後先考えず勢いで行動する、パッションあふれる女性。仕事とは責務であり、世界に対して誓う担うべき責任と義務であると神聖視している。

失踪した妹・セラフィナを探すため、ファルジーン島へ向かう。戦争孤児として育った過去から、絶望に本当に効く薬はないことを知っており、自らの足で歩いていくことの大切さを理解している。

義を通すことを重んじ、巡礼者が牢に入れられた時は、同じ牢に入り、義を通す姿勢を見せる。深きものへの特別な警戒心を持つが、恐怖に負けずに戦う。

### 特徴的な言動

- 「この世はパッションよ」
  → 勢いで行動する姿勢

- 「大事の前の小事よ」
  → 妹の失踪を知ると、後ろを振り返らずに行動する

- 「仕事とは責務であり、世界に対して誓う担うべき責任と義務である」
  → 責務への執着

- 「おまえら働け」
  → 責務を果たしていない人間への怒り

- 「人は、食べるものを選べる。食べ物が豊富にあるときは、何を口にするかを選ぶことができるのだ」
  → 人間の矜持について

### 戦闘スタイル

ハルバードを主な武器として使用する前衛戦士。前衛として味方を守りながら戦う。イルメイターの神聖魔法を使用し、仲間を支援することもできる。

## 主要NPC

### センサ船長

ヘイゼルス・フォリー号の船長。ノクシルの旧友で、2年ぶりの再会を果たす。若くして船長の座に就いた有能な人物。

### ネギ

シュブ＝ニグラスの信者。巡礼者を召喚した少女。ファルジーン島で行方不明になっている。

### コーラリア

シャルルの旧友。深きものに変容してしまった過去を持つ。

### セラフィナ

シャルルの妹。失踪中。シャルルが探している。

### ルンジャタ

ゴート教団の高司祭。失踪中。シャルルの過去と深い関係がある。

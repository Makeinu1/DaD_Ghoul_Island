# D&D 食屍鬼島キャンペーン - リプレイ小説プロジェクト

D&D 5e版とクトゥルフの呼び声TRPGを混ぜたキャンペーン「食屍鬼島」の、11回分のセッション記録から「リプレイ小説」を作成するプロジェクトです。

## 🎲 キャンペーン情報

- **DM**: モクモク
- **プレイヤー**:
  - PL1（チョモラン）: 巡礼者
  - PL2（タツヤ）: ノクシル・グリムホルト
  - PL3（まけいぬ）: シャルル・ビアトリス
- **システム**: D&D 5e/2024 + クトゥルフの呼び声TRPG（混合ルール）
- **セッション**: 第1回～第11回（2025年8月～12月）

## ⭐ データの信頼性レベル

### 最優先: 一次データ
1. **04_プレイヤーログ/** - 実際のプレイログ（最も信頼できる）
2. **07_セッション音声ログ/** - 音声文字起こし（台詞抽出用）
3. **03_プレイヤー設定/シャルルストーリー/** - 公式背景設定

### 参考: 二次データ（誤りの可能性あり）
1. **05_リプレイ/** - 他LLM生成リプレイ（要検証）
2. **08_LLMによるキャラ評/** - キャラクター分析（補助資料）

## 🎯 プロジェクトの目的

1. **正確性**: 一次データに基づいた事実確認済みのリプレイ
2. **読みやすさ**: 読み物として面白い構成
3. **内輪感**: プレイヤーとDMが楽しめる雰囲気

## 📝 リプレイ小説作成の基本方針

### 情報の優先順位
```
プレイヤーログ > 音声ログ > 公式設定 > 参考資料
```

### 執筆時の注意点
✅ **DO（推奨）**
- プレイヤーログを最優先で参照
- 複数ログで事実確認
- 音声ログから特徴的な台詞を抽出
- 背景設定を活用して深みを出す

❌ **DON'T（非推奨）**
- 05_リプレイ/を鵜呑みにしない
- 一次データと矛盾する情報を採用しない
- プレイヤーログにない出来事を創作しない

## 📖 主要キャラクター

### プレイヤーキャラクター（PC）
- **巡礼者**: シュブ＝ニグラスの信者。異形だが邪気がない
- **ノクシル・グリムホルト**: エルフのレンジャー。二刀使い、クールで寡黙
- **シャルル・ビアトリス**: イルメイターの信徒。正義感が強く、パッション溢れる

### 重要NPC
- **センサ船長**: ヘイゼルス・フォリー号の船長。ノクシルの旧友
- **ネギ**: シュブ＝ニグラスの信者。巡礼者を召喚した少女
- **コーラリア**: シャルルの旧友。深きものに変容
- **ルンジャタ**: ゴート教団の高司祭。失踪中
- **セラフィナ**: シャルルの妹。失踪中

## 🎭 物語の舞台

- **レサンテ**: 衰退した港町（出発地点）
- **ファルジーン島**: 「海の道標」として知られる島
- **ファルジーン市**: 島の首都。光の女神の巨大な像がある
